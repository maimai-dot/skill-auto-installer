# Claude Code 神技：说人话就能自动安装技能，这个开源项目让你的 AI 编程效率翻倍

## 痛点

用 Claude Code 的人都知道，它有一个强大的技能（Skills）系统——处理 PDF、生成 Excel、设计前端页面、构建 MCP 服务器……各种技能应有尽有。

但问题来了：**你得先知道有这些技能，还要手动安装它们。**

"我想生成一份 PDF 报告" → 你需要知道有 `pdf` 这个技能 → 手动 `/plugin install pdf` → 再回来让 Claude 帮你做。

这就像你去餐厅，服务员告诉你"我们的菜都在后厨，你知道菜名才能点"。体验割裂。

## 解决方案：Skill Auto-Installer

我做了一个开源项目，零配置，一行命令，让 Claude Code 听懂人话：

> **你说需求，它自动找到技能、安装、然后帮你完成任务。**

```
你: "帮我把这份数据生成一个 PDF 报告"

Claude 内部自动完成:
  [意图分析]  检测到"PDF" → 需要 pdf 技能
  [安装检查]  pdf 未安装
  [市场搜索]  在技能市场中找到 pdf/
  [自动安装]  复制到 ~/.claude/skills/pdf
  [调用执行]  使用 pdf 技能生成你的报告

你只看到: 一份精美的 PDF 报告 ✨
```

## 技术架构

采用**双层匹配**策略，兼顾速度和精度：

| 层级 | 机制 | 速度 | 精度 | 作用 |
|------|------|------|------|------|
| Hook 层 | 正则关键词匹配 | 毫秒级 | 中 | 预扫描，提前安装明显匹配的技能 |
| Skill 层 | Claude 语义理解 | 秒级 | 高 | 深度分析意图，精确匹配并补装 |

### 工作流程

```
用户自然语言输入
       │
       ├──→ [Hook] onUserPromptSubmit.js
       │     17组正则→技能映射，毫秒级预装
       │
       └──→ [Skill] skill-auto-installer.md
            Claude 深度语义分析，精确匹配
            安装后立即调用技能完成任务
```

## 安装

```bash
# 方式一：作为全局技能
git clone https://github.com/maimai-dot/skill-auto-installer.git
cp -r skill-auto-installer/.claude/skills/skill-auto-installer.md ~/.claude/skills/

# 方式二：配合 Hook（推荐，获得双层加速）
# 在 .claude/settings.json 添加 onUserPromptSubmit hook
```
