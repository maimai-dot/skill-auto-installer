链接：https://www.reddit.com/r/ClaudeAI/submit

**Title:**
```
I built a meta-skill for Claude Code that auto-installs skills based on what you say
```

**Text:**
```
You say "generate a PDF report" — it automatically finds the pdf skill, installs it, and uses it. No /plugin install needed.

## Architecture
Dual-layer matching: Hook (regex pre-scan, ~ms) + Skill (semantic analysis, ~s).
The hook pre-installs obvious matches before Claude even processes the prompt.
The skill handles ambiguous cases with deep understanding.

## Supported (17 intent domains)

Would love feedback and stars!
```
